# teach-la-website

(WIP) a quick frontend for ACM Teach LA!

To take a peek, first do:

```bash
gem install bundler
```

Then, run:

```bash
bundle
```

This should install all of our RubyGems dependencies!

Finally, run:

```bash

$ bundle exec jekyll serve
```
